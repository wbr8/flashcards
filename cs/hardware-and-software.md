Q: What is hardware?
A: Hardware is all the electrical or electro-mechanical parts of a computer, and its input, output and storage devices.

Q: What is the name given to the electrical or electro-mechanical parts of a computer, and its input, output and storage devices?
A: Hardware.

Q: What is software?
A: The programs that are written to make computers function.

Q: What is the name given to the programs that are written to make computers function?
A: Software.

Q: What is system software?
A: The software needed to run the computer's hardware and application programs.

Q: What is the name given to the software needed to run the computer's hardware and application programs?
A: System software.

Q: Name the four categories of system software.
A: 
- Operating system 
- Utility programs
- Libraries
- Translators

Q: Explain the functions of an operating system.
A: 
- To allocate memory to processes.
- To handle interrupts.
- To hide the complexities of the hardware from the user.
- To schedule processes.

Q: Describe some purposes of utility software.
A: 
- To backup files.
- To defragment hard drives.
- To compress data.
- To check for viruses.

Q: What are libraries?
A: Useful, frequently used programs that are pre-compiled and can be run as needed.

Q: Name the three types of translator.
A: 
- Compiler
- Interpreter
- Assembler

Q: What is the purpose of a translator?
A: To translate program code into machine code.

Q: Name the three categories of application software.
A: 
- General-purpose software 
- Special-purpose software 
- Bespoke software 

Q: What is general-purpose software?
A: Software that can be used for many different purposes, e.g. word processors, spreadsheets, graphics packages.

Q: What is special-purpose software?
A: Software that performs a single specific task or set of tasks.

Q: What is bespoke software?
A: Software that is custom made to satisfy some particular requirements.

Q: What are the two low-level programming languages?
A: 
- Machine code 
- Assembly language
